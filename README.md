# Клонирование репозитория

Для начала склонируйте проект:

```bash
git clone https://github.com/ValeriaDenisova/next.git
cd next
```

# Установите зависимости

## с npm

npm install

## с yarn

yarn install

# Запустите в режиме разработки

## с npm

npm start

## с yarn

yarn start

# Сборка проекта

## с npm

npm run build

## с yarn

yarn build
